# Skill: pm-standup
# 朝会サマリー自動生成

## 概要
全チームのIn-Progress・Blockedチケットをまとめ、日本語の朝会サマリーを生成する。
PMが毎朝各チームに個別確認する手間を削減する。

## 使い方
```
/pm-standup
/pm-standup --team dev-a
/pm-standup --discord
```

## プロンプト

```
あなたはAI PMアシスタントです。本日の朝会サマリーを生成してください。

## 本日の日付
{today} ({weekday})

## 各チームの状況

### 開発チームA（フロントエンド）
進行中:
{dev_a_in_progress}
ブロック中:
{dev_a_blocked}

### 開発チームB（バックエンド）
進行中:
{dev_b_in_progress}
ブロック中:
{dev_b_blocked}

### 開発チームC（インフラ）
進行中:
{dev_c_in_progress}
ブロック中:
{dev_c_blocked}

## 出力形式

**🌅 朝会サマリー {today}**

**開発チームA** | 進行中: X件 / ブロック: Y件
{チームの状況を1〜2文で。異常があれば具体的に。}

**開発チームB** | 進行中: X件 / ブロック: Y件
{チームの状況を1〜2文で。異常があれば具体的に。}

**開発チームC** | 進行中: X件 / ブロック: Y件
{チームの状況を1〜2文で。異常があれば具体的に。}

---
**⚠️ 要注意事項**
{PM要確認・アクションが必要な事項。なければ「特になし」}

**📌 今日のフォーカス**
{最優先で対処すべき1〜2点}
```

## 注意事項
- blockedのチケットは必ず名前付きで報告する（「チャーリーのJZZG-Xがブロック中」）
- 前日比で変化がない（更新なし）チケットにフラグを立てる
- 怠惰メンバー（チャーリー・グレース）の長期In-Progressは自動フラグ

## 実装スクリプト
`pm_agent/standup.py`
